# qi
非侵入式App性能采集

# 下载
[下载](https://github.com/oneym/qi/releases/latest)地址

# 使用方法
```shell
./qi -p <pid>
```

命令帮助
```shell
  -h, --help                 show this help
  -p, --pid int              target pid
      --qi-server-port int   qi server port (default 54168)
  -v, --version              qi version

```
